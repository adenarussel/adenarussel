EnGuage

Web App

EnGuage is a web application that was built as part of the Make-a-thon for Affectiva's Summer 2020 trainee program (apprenticeship). Our goal was to build a Zoom extension which allowed presenters and attendees with a gauge for different metrics. The metrics, engagement and attention were from the Affectiva SDK. However, we developed our own confusion metric based on facial landmarks and expressions. It was built using JavaScript. We presented the web app with a demo and the site is still live so one can demo it themselves!
